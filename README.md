# C++ Dynamic Array Minimum Finder

This program creates a dynamic array based on user input, stores values, and finds the smallest value in the array using a function.

This project was completed as part of a Computer Engineering course at the University of North Texas.

## Features
- Dynamic memory allocation using new and delete
- Pointer-based array access
- User input handling
- Function to compute minimum value

## What I learned
- How to use pointers and dynamic arrays
- How memory allocation works in C++
- How to pass arrays to functions
- How to separate logic using functions

## How to run
Compile and run using a C++ compiler:

g++ dynamic_array_min.cpp  
./a.out
